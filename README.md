# GoScrum
## About
GoScrum is an App similar to Trello, where users can register and create new teams to work under the scrum methodology.

## Getting Started 
Clone respository:
```
git clone https://github.com/lucianoumc1/GoScrum.git
```

Install the dependencies:
```
npm install
```

Run in dev mode.
```
$ npm run dev
```
